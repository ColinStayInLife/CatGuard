## 执行摘要

### 项目概述

CatGuard 是一款专注解决"猫咪乱尿问题"的智能硬件产品，采用边缘AI技术实现实时行为识别与声波干预，帮助养猫家庭解决乱尿困扰。

### 核心数据

| 指标 | 数值 |
|------|------|
| 目标市场 | 5-15 亿元 |
| 产品定价 | ¥499-699 |
| 目标销量 (6个月) | 3,000-5,000 台 |
| 预期营收 (6个月) | ¥150-300万 |
| 毛利率 | 40-50% |
| 团队需求 | 5-8 人 |

### 核心优势

1. **首创定位**: 首款专注"乱尿预防"的智能硬件
2. **技术壁垒**: 边缘AI + 3秒预警 + 声波干预
3. **隐私保护**: 无视频上传，本地推理
4. **价格优势**: 仅为智能猫砂盆的 1/3

= TAM × 渗透率

## 第三章：产品与技术

### 3.1 产品概述

#### 产品定义

CatGuard 是一款基于边缘AI的猫咪行为监测设备，通过实时识别猫咪蹲姿，在乱尿发生前 3 秒发出声波预警，帮助猫咪建立正确如厕习惯。

#### 核心功能

| 功能 | 描述 | 技术实现 |
|------|------|----------|
| **实时监测** | 24小时监测猫咪行为 | 摄像头 + 边缘AI |
| **蹲姿识别** | 3秒识别排尿前兆 | YOLOv8 分类模型 |
| **声波干预** | 发出特定频率声波 | 超声波/蜂鸣器 |
| **行为记录** | 记录猫咪行为日志 | 本地存储 + 云端同步 |
| **异常提醒** | 手机APP推送警报 | WiFi + 消息推送 |
| **隐私保护** | 视频不上传云端 | 边缘计算架构 |

### 3.3 AI模型

#### 模型规格

| 项目 | 规格 |
|------|------|
| **模型架构** | YOLOv8s-cls (分类) |
| **参数量** | 5.1M |
| **FLOPs** | 12.4G |
| **输入尺寸** | 224×224 RGB |
| **输出类别** | 2类 (Positive/Negative) |
| **训练数据** | 600+ 标注图像 |
| **准确率** | 87.3% |
| **召回率** | 100% (Positive类) |

#### 模型优化

| 优化方法 | 效果 |
|----------|------|
| **INT8量化** | 模型 19MB → 5MB |
| **RKNN转换** | 适配RV1126 NPU |
| **分辨率降低** | 224→192 可选 |

#### 推理性能

| 平台 | 模型 | FPS | 备注 |
|------|------|-----|------|
| J1900 (CPU) | YOLOv8s-cls FP32 | 15 | 当前方案 |
| RV1126 (NPU) | YOLOv8s-cls INT8 | 30-50 | 推荐方案 |
| RV1126 (NPU) | YOLOv8n-cls INT8 | 60-80 | 轻量方案 |
